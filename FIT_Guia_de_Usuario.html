Introdução

Bem-vindo ao guia de usuário do FIT Traffic Lights for Choices, uma ferramenta inovadora que adiciona um sistema visual de "semáforo" aos elementos de formulário em páginas web. Esta ferramenta permite que você classifique visualmente opções em três estados (vermelho, amarelo e verde) sem interferir no funcionamento original dos formulários.

Desenvolvida para auxiliar na análise e tomada de decisão, esta ferramenta é especialmente útil em situações onde você precisa classificar ou priorizar múltiplas opções em formulários complexos, questionários ou listas de verificação.

O FIT Traffic Lights for Choices é uma solução temporária e não invasiva que pode ser facilmente adicionada a qualquer página web, permitindo uma camada adicional de informação visual sem modificar o comportamento original dos elementos de formulário.

Funcionalidades Principais

Sistema de 3 Estados

Adiciona indicadores visuais tipo "semáforo" ao lado de checkboxes e radio buttons, permitindo classificar opções em três estados distintos: Vermelho (FALSE), Amarelo (INDETERMINATE) e Verde (TRUE).

Painel de Controle

Interface flutuante posicionada no canto superior direito da página, com botões para alternar a visibilidade dos indicadores (ON/OFF) e remover completamente a ferramenta quando não for mais necessária.

Detecção Automática

Processa automaticamente todos os checkboxes e radio buttons existentes na página e utiliza MutationObserver para detectar e processar novos elementos adicionados dinamicamente.

Design Responsivo

Interface totalmente adaptável a diferentes tamanhos de tela, com suporte específico para dispositivos móveis e telas sensíveis ao toque, garantindo uma experiência consistente em qualquer dispositivo.

Não Invasivo

Funciona como uma camada visual adicional sem interferir no comportamento original dos elementos de formulário, preservando a funcionalidade original da página.

Remoção Completa

Quando não for mais necessário, o sistema pode ser completamente removido da página, restaurando-a ao seu estado original sem deixar resíduos.

Como Usar

Adicionar o Script ao Navegador

Para adicionar o script ao navegador há uma página de instalação que facilitará o processo de geração e instalação do bookmarklet. Visite-a no seguinte endereço: https://magasine.github.io/FIT_TrafficLightsForChoices/

Verificar a Ativação

Após adicionar o script, uma mensagem de boas-vindas será exibida confirmando a ativação do sistema. Você também verá um painel de controle no canto superior direito da página e indicadores visuais ao lado de cada checkbox e radio button.

Utilizar os Indicadores Visuais

Clique nos indicadores visuais ao lado de cada checkbox ou radio button para alternar entre os três estados disponíveis (vermelho, amarelo e verde). Isso permite classificar visualmente as opções sem alterar o estado real dos elementos de formulário. Ao clicar em um indicador, ele se tornará ativo e mudará de cor, fornecendo um feedback visual claro sobre o estado selecionado. Esta funcionalidade é especialmente útil quando você precisa classificar múltiplas opções em diferentes categorias durante processos de análise ou tomada de decisão. As classificações de estados são (F)alse, (I)ndeterminate e (T)rue.

Controlar a Visibilidade

Use o botão ON/OFF no painel de controle para alternar a visibilidade dos indicadores visuais. Isso permite ocultar temporariamente os indicadores sem removê-los completamente da página. Esta funcionalidade é particularmente útil quando você precisa visualizar o formulário em seu estado original para comparação ou quando deseja reduzir temporariamente os elementos visuais na tela para focar em outras partes do conteúdo. Os indicadores permanecerão na página, mantendo seus estados, e poderão ser exibidos novamente com um simples clique no botão ON.

Quando não precisar mais do sistema de marcação visual, clique no botão × no painel de controle. Uma mensagem de confirmação será exibida e, após confirmar, o sistema será completamente removido da página, restaurando-a ao seu estado original. Este processo de remoção é completo e limpo, eliminando todos os elementos visuais adicionados, desconectando os observadores de eventos e removendo quaisquer modificações feitas à página. É importante notar que todas as marcações visuais serão perdidas após a remoção, portanto certifique-se de ter concluído sua análise ou tomado nota das classificações importantes antes de remover o sistema.

Sistema de Estados

O FIT Traffic Lights for Choices utiliza um sistema de três estados, representados por cores diferentes, para permitir a classificação visual de opções em formulários web:

Vermelho (FALSE)

Representa uma opção rejeitada, incorreta ou que não atende aos critérios desejados. É útil para marcar itens que devem ser desconsiderados ou que necessitam de correção.

Amarelo (INDETERMINATE)

Representa uma opção em análise, parcialmente correta ou que requer mais informações. É útil para marcar itens que precisam ser revisados posteriormente ou que estão em consideração.

Verde (TRUE)

Representa uma opção aceita, correta ou que atende plenamente aos critérios desejados. É útil para marcar itens aprovados ou que foram confirmados como válidos.

Importante ressaltar que estes estados são puramente visuais e não afetam o comportamento real dos elementos de formulário. Isso significa que você pode usar este sistema para classificar visualmente as opções sem interferir no processamento do formulário original.

Casos de Uso

O FIT Traffic Lights for Choices é uma ferramenta versátil que pode ser aplicada em diversos contextos. Abaixo estão alguns cenários onde esta ferramenta pode ser particularmente útil:

Análise de Formulários

Ideal para formulários complexos com muitas opções que precisam ser classificadas ou priorizadas. Permite marcar visualmente itens para revisão posterior sem alterar as seleções originais.

Tomada de Decisão

Facilita a classificação de opções em "não aceitas" (vermelho), "em consideração" (amarelo) e "aceitas" (verde), auxiliando em processos de seleção com múltiplos critérios.

Revisão de Checklists

Permite adicionar um nível adicional de informação a listas de verificação, indicando status além do simples "marcado/não marcado", como "verificado e aprovado", "verificado e reprovado" ou "precisa de verificação adicional".

Ambientes Educacionais

Pode ser usado por professores para avaliar respostas de alunos, facilitando a marcação de questões corretas, parcialmente corretas ou incorretas durante a correção de testes online.

Auditorias e Inspeções

Auxilia profissionais que precisam classificar itens durante inspeções, permitindo marcar itens que necessitam de atenção especial, estão em conformidade ou apresentam problemas.

Acessibilidade

Fornece feedback visual adicional para usuários que se beneficiam de indicações visuais claras. Os indicadores de cores podem ajudar pessoas com certas necessidades cognitivas a processar melhor as informações.

Perguntas Frequentes

O script modifica os dados do formulário original?

Não. O FIT Traffic Lights for Choices funciona como uma camada visual adicional e não interfere no comportamento ou nos dados dos elementos de formulário originais. Os estados visuais (vermelho, amarelo e verde) são independentes do estado real dos checkboxes e radio buttons.

Os indicadores visuais são salvos quando a página é recarregada?

Não. Os indicadores visuais são temporários e existem apenas durante a sessão atual. Se a página for recarregada, os indicadores voltarão ao estado inicial. Esta ferramenta é projetada para uso temporário durante a análise de formulários.

É possível usar em qualquer site?

Sim. Por ser um script injetável, o FIT Traffic Lights for Choices pode ser usado temporariamente em qualquer página web que contenha elementos de formulário como checkboxes e radio buttons. No entanto, alguns sites com medidas de segurança rigorosas podem bloquear a execução de scripts externos.

O script funciona com elementos de formulário adicionados dinamicamente?

Sim. O script utiliza MutationObserver para detectar e processar novos elementos de formulário adicionados dinamicamente à página após a inicialização do sistema.

É possível personalizar as cores ou rótulos dos estados?

Na versão atual (v20250530), as cores e rótulos dos estados são predefinidos. Para personalizações, seria necessário modificar o código-fonte do script, alterando os valores no objeto CONFIG.STATES.

O script é compatível com dispositivos móveis?

Sim. O FIT Traffic Lights for Choices foi desenvolvido com design responsivo e inclui adaptações específicas para dispositivos móveis e telas sensíveis ao toque, garantindo uma experiência consistente em qualquer dispositivo.

Solução de Problemas

O script não está sendo aplicado a alguns elementos de formulário

Verifique se os elementos são checkboxes ou radio buttons padrão. O script foi projetado para funcionar com elementos HTML padrão e pode não reconhecer componentes personalizados que simulam o comportamento de checkboxes ou radio buttons.

Mensagem "Traffic Lights For Choices já está ativo" aparece no console

O script já está em execução na página. Para reiniciar, primeiro remova a instância atual clicando no botão × no painel de controle e, em seguida, execute o script novamente.

O painel de controle está sobrepondo elementos importantes da página

O painel de controle é posicionado no canto superior direito da página. Se estiver interferindo com elementos importantes, você pode temporariamente ocultar o painel modificando seu CSS através do console do navegador ou remover completamente o script quando não for mais necessário.

Os indicadores visuais desapareceram após interagir com a página

Verifique se o botão no painel de controle está em "ON". Se estiver em "OFF", os indicadores estão ocultos mas ainda presentes. Clique no botão para alternar para "ON" e torná-los visíveis novamente.

O script está causando conflitos com outros scripts da página

O FIT Traffic Lights for Choices foi projetado para minimizar conflitos, mas em alguns casos pode interferir com outros scripts. Se notar comportamentos inesperados na página após adicionar o script, remova-o completamente clicando no botão × no painel de controle.
